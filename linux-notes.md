Basic linux commands - 
ls , ls -la , ls -l , ls /etc   — these are used to see directory and file
cd   —this is used to change directory,  
cd ..    —this is used to go one directory back.
cd ../..      — this is used to go two directory back. 
cd ~         — this is used to go directly into home directory.
cd /    — this is used to go directly into root directory.
pwd     — this is used to check current location.
mkdir   — this is used to creat directory.
mkdir -p   — this is used to creat nested directory.
touch    — this is used to creat file
rm       — this is used to delete file
rm -r    — this is used to delete directory
rm -rf /   — this is used to to delete everything
cp         — this is used to copy file 
cp -r      — this is used to copy whole directory
mv         — this is used to rename and move directory and file 
cat      — this is used to watch file contant 
echo "text"    — this is used to write text
echo "text" > file name       — this is used to write contant inside any file
echo "text" >> file name      this is used to write more contant inside any file with existing contant
